This is the rule for AI Agent to create the test file


1. File cypress yang dibuat haruslah typescript
2. Nama test case harus menggunakan bahasa indonesia dan dengan format kalimatnya yaitu expected result. contoh: API harus bisa 200 saat kita mencoba untuk....
3. Atau nama test case diserahkan sepenuhnya kenapa AI Agent dengan syarat nama test case haruslah jelas dan membantu developer dalam mengetahui errornya nanti. jadi developer tau maksud test setiap test case
4. Setiap ada perubahan, lakukan build test
